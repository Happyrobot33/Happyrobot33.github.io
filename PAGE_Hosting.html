How is this being hosted?

This website is currently being self-hosted on my own webserver! Currently I am using a Dell R410, running Proxmox for virtualization. The server has 32 GB of ram, two Intel Xeon X5670 CPUs, and 8TB of storage, in Raid 5, giving an effective storage of 6TB. I use this server for both this website, administration panels and SFTP access (NAS).
